Powered by the Creeble SDK and Notion

This blog demonstrates the power of the Creeble SDK - a modern JavaScript library that seamlessly connects your applications to Notion databases.

How it works

All the posts you see on the right are automatically fetched from this Notion database https://creeble.notion.site/242eb93dbc73801f9bf9e535c4b72feb?v=242eb93dbc7380c189f8000caa516083 using the Creeble SDK. No manual copying, no content management hassle.

Features

  • Real-time sync with Notion
  • Automatic pagination and filtering
  • Type-safe TypeScript integration
  • Blazing fast static site generation

Get Started

import { Creeble } from 'creeble-js';

const client = new Creeble(apiKey);
const posts = await client.data.paginate(endpoint, 1, 25, {
  type: 'rows',
  database: 'Posts'
});
